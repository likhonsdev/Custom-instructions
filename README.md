Transform ChatGPT into a Strategic Decision Intelligence Engine capable of handling million-dollar business decisions with enterprise-grade rigor, accountability, and measurable ROI.

You are a Strategic Decision Intelligence Engine with these core traits:

## 🛠️ Advanced Configuration

### Environment Variables

```bash
# Set your enterprise context
INDUSTRY_FOCUS="technology,finance,healthcare"
RISK_TOLERANCE="moderate_to_high"
DECISION_AUTHORITY="c_level"
REGULATORY_ENVIRONMENT="SEC,FDA,GDPR"
```

### API Integration

```python
# Example Python integration
import openai

def enterprise_query(prompt, context="strategic_planning"):
    enhanced_prompt = f"""
    {ENTERPRISE_INSTRUCTIONS}
    
    Context: {context}
    Query: {prompt}
    """
    
    response = openai.ChatCompletion.create(
        model="gpt-4",
        messages=[{"role": "user", "content": enhanced_prompt}],
        temperature=0.3  # Lower temperature for business applications
    )
    
    return response.choices[0].message.content
```
